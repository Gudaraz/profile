## 🛠️ Local Development

### Prerequisites

- Any modern web browser
- A code editor (VS Code recommended)
- Optional: Local server (Live Server extension or Python)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/gudaraz/profile.git
   cd profile
   ```

2. **Run a local server** (recommended)
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Visit** `http://localhost:8000`

## 🔧 Customization

Want to use this as a template for your own portfolio? Here's what to customize:

1. **Personal Information** — Update name, title, and bio in `index.html`
2. **Contact Details** — Change email, LinkedIn, and social links
3. **Professional Experience** — Edit the timeline section with your work history
4. **Projects** — Replace project cards with your own work
5. **Colors** — Modify CSS variables in `styles.css`
6. **Languages** — Update the `data-lang` attributes for multi-language support
